# Database-Programming-Assignment-30583-2025-ALSHEAB


Project Overview:

​This project involves designing a relational database for a Sales Management System. It demonstrates data modeling, SQL query implementation, and analytical insights.

Key Features:

​Database Design: Normalized tables for Customers, Products, and Orders.

​Advanced SQL: Implementation of CTEs and Window Functions for data analysis.

​Data Insights: Descriptive, Diagnostic, and Prescriptive analysis of sales performance.

Analysis Summary:

​Descriptive: Analyzed 7 orders across 5 unique customers.

​Diagnostic: Identified high-demand products (Laptops, Keyboards) through aggregation queries.

​Prescriptive: Recommended stock optimization and customer loyalty strategies to enhance business growth.
